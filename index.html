How it works

What you're looking at

Every orbit is defined by a handful of numbers called orbital elements. This tool turns those numbers into a 3D trajectory using the same two-body + J2 physics as the underlying Python engine — recomputed live as you drag.

Orbital elements → position

The sliders set the orbit's size, shape, and orientation. The app converts them into 3D coordinates the same way real flight software does.

The J2 effect

Earth isn't a perfect sphere — its equatorial bulge slowly swivels the orbit plane. Toggle J2 to see the orbit precess over time.

Real satellites

The preset chips load actual orbital parameters for the ISS and others, so you can see how their real orbits are shaped.
